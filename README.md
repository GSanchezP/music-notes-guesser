# Music Notes Guesser

A Vue 3 app that plays musical notes and lets you guess which one you heard.

## How to play

1. Click **Play Note** to hear a random note (C, D, E, F, G, A, or B in octaves 2–6).
2. Press the button for the note you think you heard.
3. See if you were correct and the actual note + octave.
4. A new note plays automatically—guess again!

## Run locally

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

## Build

```bash
npm run build
```
